# Codefun Auto Submit Tool
Đây là tool dùng để auto submit cho web codefun.vn

Trong đây có tầm 420 solutions bài giải

Vì codefun cooldown 1p30s nên tool này được làm ra để chạy auto cho bạn

## Cách chạy
Bước 1: Chạy file *run.bat*

Bước 2: Đăng nhập web codefun

Bước 3: Treo máy tới khi chạy xong (không đóng chrome mà script tạo nếu ko sẽ bị lỗi và không được bấm nút window controls của chrome, chỉ được minimize ẩn nó xuống taskbar)
